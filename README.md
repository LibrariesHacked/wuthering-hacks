# Wuthering Hacks

A data dashboard done from Newcastle public libraries open data.

## Data definitions

### Libraries

- Publisher
- Branch
- Address
- UPRN
- Northing
- Easting
- Latitude
- Longitude
- BranchCategory
- OpeningHours


### Library members

- Publisher
- Branch
- Age
- MemberCategory

### PC Usage

- Publisher
- Branch
- Month

### Visits

- Publisher
- Branch
- Month


### Issues

- Publisher
- Month
- Branch
- ItemCategory
